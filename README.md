# Business Intelligence Dashboard

A React-based business intelligence dashboard for Myanmar businesses with sales analytics and profit/loss reporting.
Demo url: [https://business-dashboard-teal.vercel.app/](https://business-dashboard-teal.vercel.app/)

## Features

- Sales dashboard with key metrics and charts
- Complete P&L reports following accounting standards
- Top performing products visualization
- CSV export for financial data

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build
```

## Technology Stack

- React 19.2.3 with TypeScript
- Vite build tool
- Recharts for data visualization
- Tailwind CSS styling
